# Emotion Recognition Project

This project detects human emotions from facial expressions using
Deep Learning and OpenCV. It supports real-time emotion detection
through a webcam.

## How to Run

1. Open this folder in VS Code
2. Create and activate virtual environment
3. Install requirements:
   python -m pip install -r requirements.txt
4. Train the model:
   python train.py
5. Run webcam:
   python camera.py

Press Q to quit webcam.
